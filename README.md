# gpui-play

A repository of experiments using the Zed GUI framework gpui

Most useful perhaps is the documentation. I've built this with Claude to get an index for myself so I have some documentation available to know how to use GPUI. I didn't find docs elsewhere, so it might be generally useful for anyone else aiming to use GPUI.

The two places to start are [an_overview.md](https://github.com/puckdoug/gpui-play/blob/main/docs/gpui-usage/an_overview.md) for a general overview of how GPUI works and [an_index.md](https://github.com/puckdoug/gpui-play/blob/main/docs/gpui-usage/an_index.md) for an index into the documentation for all elements I've been able to discover. If you find anything wrong or anything I've missed, please do open a PR to add it here.
